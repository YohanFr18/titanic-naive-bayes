## Sobre
Este projeto implementa um classificador Naive Bayes para prever a sobrevivência de passageiros do Titanic com base em características como idade, sexo e classe social.

## Tecnologias
- Python 
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
